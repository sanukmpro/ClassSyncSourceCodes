# ClassSync – Student's Study Materials Management System

ClassSync is a cross‑platform application designed to centralize and streamline the management of academic study materials for students and teachers.  
This repository contains the **main source code of the app**, including mobile (Flutter) and web versions, backend integrations, and deployment scripts.

---

## 📌 Features
- **Role-based Authentication**: Secure login for students and teachers.
- **Materials Management**: Upload, categorize, and moderate notes, assignments, and summaries.
- **Discovery Tools**: Advanced search and filtering by subject, course, and tags.
- **Communication**: One-to-one chat and community forums.
- **Gamification**: Honor scores, badges (Silver, Gold, Platinum), leaderboards, and certification for top contributors.
- **Notifications & Analytics**: Alerts for approvals, feedback, and usage reports.

---

## 🛠️ Tech Stack
- **Frontend (Mobile)**: Flutter (Dart)
- **Frontend (Web)**: HTML5, CSS, JavaScript
- **Backend Services**: Firebase Authentication, Firebase Realtime Database, Firebase Cloud Functions
- **Storage**: Firebase Storage + Cloudinary (for media files)
- **Deployment**:
  - GitHub Pages – Landing page with access to web version, download links, and system highlights
  - Firebase Hosting – Web version of the app
- **Version Control**: Git + GitHub

## 🚀 Deployment
- **Landing Page**: Hosted on GitHub Pages – provides access to the web version and download links for the mobile app.
- **Web App**: Deployed via Firebase Hosting for secure and scalable access.
- **Mobile App**: Built with Flutter, installable on Android (min SDK 21) and iOS.
